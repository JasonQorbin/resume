Corporate Finance Modelling Tool

Overview

In the world of corporate finance it's common to use mathematical models as tools to develop a view of the future state of a potential target/investee company. I was the lead developer of an Excel & VBA-based modelling tool that was used by around 350 people on a daily basis.

Compliance alongside freedom

The users of the tool had previously created bespoke Excel models for every transaction. However, the goal of the tool was to lock down the critical portions and more complex calculations to reduce human-error and increase the reader's confidence in the voracity of the final report.

Allowing the users to apply flexibility when the situation called for it required a hybrid solution. The critical portion of the tool remain permanently locked the model silently switches between a locked state and an unlocked as the user moves between the sensitive and free-form part of the tool.

Flexibility

Every business is different so the tool needed to have the ability to stretch and to meet the needs of the task at hand. The various submodules of the model were designed to be modular and tileable. Then macros were to built to reinitialise each submodule as needed.

Alert System

Despite automating most of the calculations a user could still feed the model an impossible combination of data leaving the model in an inconsistent state. An "Alert System" was designed to give the user a birds-eye view of any issues they need to address.

Alerts are dynamically added and removed as the user adds extra modules to the model.
